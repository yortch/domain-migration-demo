# E-commerce Web Application

## Setup Instructions

### Environment Variables
```
API_ENDPOINT=https://api.old.com/v1
DB_HOST=db.old.com
DB_USER=legacy_admin
API_TOKEN=your_api_token
```

### API Documentation
- Main API: https://api.old.com/docs
- Auth Service: https://auth.old.com/docs
- Support Portal: https://support.old.com/api-docs
